# WutheringWavesSteamHelper 项目结构说明

## 文件夹组织

### Assets/
资源文件夹，存放应用程序使用的各种资源文件

- **Icons/** - 存放应用程序图标文件（.ico）
- **Images/** - 存放应用程序使用的图片资源（.png, .jpg等）

### Config/
配置文件夹，存放应用程序的配置文件

### Docs/
文档文件夹，存放项目相关的文档
- README.md - 项目说明文档
- CHANGELOG.md - 版本更新日志

## 使用说明

1. **添加图标**：将 .ico 文件放入 `Assets/Icons/` 文件夹
2. **添加图片**：将图片资源放入 `Assets/Images/` 文件夹
3. **添加配置**：将配置文件放入 `Config/` 文件夹
4. **更新文档**：在 `Docs/` 文件夹中编辑相关文档

注意：Assets 和 Config 文件夹中的文件会在编译时复制到输出目录。
