## 视觉风格

- **Windows 11**：通过 `DwmSetWindowAttribute` 启用 Mica 背景
- **Windows 10 降级**：普通浅色主题 `BackColor = Color.FromArgb(243, 243, 243)`
- **侧边栏**：深色 `#1E2A3A`，选中项高亮 `#3B82F6`
- **内容区**：白色半透明卡片，圆角效果
- **字体**：Microsoft YaHei UI

## 数据结构

### GameConfig.cs（新建）

```csharp
public class GameConfig
{
    public string Name { get; set; }
    public string IconResourceName { get; set; }
    public int SteamAppId { get; set; }
    public int SteamDepotId { get; set; }
    public string GameFolderName { get; set; }
    public string ExeRelativePath { get; set; }
    public string? CnLauncherExeName { get; set; }   // 国服启动器 exe 名
    public string? CnGameRelativePath { get; set; }  // 国服游戏 exe 相对路径
    public bool IsPlaceholder { get; set; }          // true = 预留，显示"敬请期待"
}
```

## 新增功能

### 打开官方启动器

- 从 `DetectCnWutheringWavesPaths()` 获取安装目录
- 拼接 `launcher.exe` 路径并 `Process.Start()`
- 找不到时提示用户手动选择
