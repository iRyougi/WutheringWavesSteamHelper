# Changelog

鸣潮 Steam 助手 — 版本更新记录

---

## v2.0.0 (2026-04-11)

### 💥 重大更新 — 全面重构

v2.0.0 是对鸣潮 Steam 助手的完整重写，从底层框架到界面设计均发生根本性变化，与 v1.x 不具备向下兼容性。

---

### 🏗️ 框架迁移

- 从 **WinForms (.NET 8)** 迁移至 **WinUI 3 / Windows App SDK**，获得原生 Fluent Design 支持
- 全新响应式布局，支持 **PerMonitorV2 DPI 自动缩放**，适配高分辨率屏幕
- 窗口尺寸调整为 1100×780，固定大小，禁止缩放

---

### 🎨 界面重设计

- 采用 **NavigationView 侧边栏导航**，主内容区与导航分离
- 顶部标题栏统一显示应用名称
- 所有卡片使用 `CardBackgroundFillColorDefaultBrush` 主题色，自动适配深色 / 浅色模式
- 应用图标统一为 `WutheringWavesSteamHelper.ico`

---

### ✨ 新功能

#### 更新检查系统（全新）

- 应用启动时**后台静默检查更新**，不阻塞主流程
- 自动探测 GitHub 连通性（3 秒超时），失败则自动切换**镜像站**，对国内用户友好
- 发现新版本时，**标题栏右上角**显示蓝色"发现新版本"提示按钮，点击跳转设置页
- 设置页内显示更新通知卡片，包含更新说明与"立即下载"按钮
- 支持 **`forceUpdate` 强制更新模式**：覆盖全屏遮罩、禁用所有导航，强制用户前往下载
- 支持**手动检查更新**按钮，含加载状态反馈

#### 版本管理

- 引入 `AppInfo.cs` 作为版本信息的**单一真相来源**，修改一处即同步所有界面显示
- `version.json` 托管于 GitHub，支持远程控制更新推送

#### 测试版渠道（开发者功能）

- 开发者模式下可开启**测试版渠道**，从指定测试分支拉取 `version.json`
- Debug 模式下更新检查走本地 `127.0.0.1:9090`，方便本地调试

#### 设置持久化

- 所有设置（开发者模式、Debug 模式、测试版渠道、语言、游戏路径等）持久化至  
  `%AppData%\WutheringWavesSteamHelper\settings.json`

---

### 🛠️ 开发者选项

- **开发者模式**：展开隐藏的调试面板
  - **Debug 模式**：更新检查使用本地服务器，方便测试
  - 显示设置文件路径，支持一键打开所在目录
- **测试版渠道**：从测试分支获取更新通知

---

### 🐛 已知问题 / 说明

- 多语言支持（繁中、英语、日语）界面已预留，功能开发中
- 本版本为 Alpha 测试阶段，如遇问题请[提交 Issue](https://github.com/iRyougi/WutheringWavesSteamHelper/issues)

## v1.1.0 (2025-03-23)

### ✨ 新功能

- 新增侧边栏导航（180px 宽，深色 #1E2A3A），内容区宽度 680px
- 窗口尺寸从 720×500 调整为 860×620
- 侧边栏底部显示当前版本号 v1.1.0
- 新增"打开官方启动器"按钮（灰蓝色），自动搜索 `launcher.exe`
- 预留碧蓝档案、尘白禁区入口（点击提示"敬请期待"）
- 新增 `GameConfig.cs`，包含 `GameConfigs.All` 静态游戏列表

### 🛠️ DPI / 多分辨率适配

- `Program.cs` 改用 `HighDpiMode.PerMonitorV2`
- `MainForm.Designer.cs`：`AutoScaleMode` 改为 `Dpi`，坐标以 96 DPI 为基准
- 新增 `app.manifest`，声明 Windows 10/11 兼容性

## v1.0.0 (2025-01-15)

### 🎉 首次发布

- 支持检测 Steam 安装路径与库目录
- 自动生成鸣潮 ACF 清单文件
- 支持从 SteamDB 获取 BuildId
- 国服游戏来源选择（官服 / 渠道服）并自动查找安装路径
- 基础日志输出，支持自动滚动
- WinForms 界面，窗口尺寸 720×500
